# Sorting
Bubble Sort: 
Time: O(n²)
Space:O(1)
Stable:Yes

Selection Sort:
Time: O(n²)
Space:O(1)
Stable:No

Insertion Sort: 
Time: O(n²)
Space:O(1)
Stable:Yes

Merge Sort:
Time:O(n log n)
Space:O(n)
Stable:Yes

Quick Sort:
Time:O(n log n) avg
Space:O(log n)
Stable:No

Heap Sort:
Time:O(log n)
Space:O(1)
Stable:No

## Imp
Merge sort = stable + extra space
Quick sort = fastest average case
Insertion sort = good for nearly sorted data
Bubble sort = worst, only for basics
